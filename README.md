# Amazon-Book-Sales-
This shows Amazon book sales perfromance from 2023-2025, highlighting top sale, and insight

## 📌 Project Overview
This project analyzes Amazon book sales data from 2023 to 2025 to identify trends, top-performing categories, and sales patterns. Microsoft Excel was used for data cleaning and preparation, while Tableau was used to create an interactive dashboard for visualization.

---

## 🎯 Objectives
- Distribution of genres of the books
- Identify top-performing books  
- Correlation between rating and the price of books  
- Distribution of reviews using price and rating

---
